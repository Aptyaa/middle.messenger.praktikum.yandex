## Яндекс Практикум Мессенджер

Учебный проект мессенджера с использованием Vite, Handlebarse, Scss, Express, TypeScript, для тестов используется mocha, chai, sinon

## Команды

Билд и запуск проекта на порту 3000

```
npm install
npm run start
```

Запуск на дев

```
npm install
npm run dev
```

Запуск тестов

```
npm test
```
